# Genesis Energy: Supply Chain Analysis and Recommendations

Dedicated portfolio page for the Genesis Energy supply chain report.

## Summary

This project analyses Genesis Energy's supply chain across fuel procurement, electricity generation,
transmission and distribution support, and retail operations. The report focuses on resilience,
renewable transition, and digital or AI-enabled improvement opportunities.
